Activity Diagram


Activity/state diagram:A system's dynamic features and components are described using an activity/state diagram. There are 4 different actors in our activity diagram.First, a patient is referred by a doctor to IPD and then patient is allowed to choose room, which will then be allocated to it by inpatient admin.After this a nurse is appointed to monitor patient's vitals. In case of any emergency nurse calls the doctor via inpatient admin. Now doctor checks the patient's vitals and if are stable, it orders discharge. If not, Medicines are prescribed. A patient is dischaged only when all bill amount is paid.
